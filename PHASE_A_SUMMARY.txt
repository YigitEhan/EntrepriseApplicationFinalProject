================================================================================
  ARCHITECTURE HIGHLIGHTS
================================================================================

MVC Pattern:
  Browser → Controller → Service → Repository → Entity → Database

Layered Architecture:
  1. Presentation Layer (Thymeleaf templates)
  2. Web Layer (Controllers)
  3. Business Layer (Services)
  4. Data Access Layer (Repositories)
  5. Domain Model (Entities)
  6. Database (H2 in-memory)

Key Design Decisions:
  - Category as Entity (not enum) for flexibility
  - BCrypt for password hashing (security best practice)
  - Session-based authentication (Phase B)
  - Spring Data JPA query derivation for filtering
  - Lombok for boilerplate reduction
  - H2 in-memory for development simplicity
